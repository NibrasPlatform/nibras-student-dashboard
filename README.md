# nibras-student-dashboard
Student dashboard for : courses, competitions, community, achievements, and AI tutor
## Routes / pages (1:1 with UI)

/login
role selector: Student / Instructor / Admin

/dashboard
KPI cards + Recent Activities + Deadlines + Progress + Achievements preview

/courses
Tabs: All / Core / Electives / Competitive + “My Courses”

/competitions
Tabs: Contests / Practice / History / Rankings

/community
Tabs: Recent / Popular / Unanswered / My Questions + tags panel + top contributors

/achievements
Tabs: Achievements / Leaderboard / Reputation

/ai-tutor
Tabs: Tutor / Recommendations / Insights / Smart Routing

/settings
This makes your codebase easy: each sidebar item = one route + one page module.

CLI Setup Guide

Install the Nibras CLI, authenticate with GitHub, and submit your first project — step by step.

0 / 10 done
01

Prerequisites

Node.js ≥ 18, npm ≥ 9, and git

You need Node.js ≥ 18, npm ≥ 9, and git before installing the CLI. Follow the steps for your OS.

Install Node.js ≥ 18 + npm

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.zshrc
nvm install --lts
nvm use --lts
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install --lts
nvm use --lts

Recommended: Use one shell and keep using it.

Open Windows Terminal and choose PowerShell. Install Node.js, then install Git. Close that terminal and open a new PowerShell window before verifying.

Open Windows Terminal and choose Git Bash. Install Node.js, then install Git. Close that terminal and open a new Git Bash window before verifying.

# Option A — winget (Windows 10 1709+ / Windows 11)
winget install OpenJS.NodeJS.LTS

# Option B — download the official .msi from nodejs.org
# (includes npm, adds Node.js to PATH automatically)

Install Git

# Xcode Command Line Tools
xcode-select --install
# Debian/Ubuntu
sudo apt-get install git

# Fedora
sudo dnf install git
winget install Git.Git

Or download the installer from git-scm.com/download/win. During setup, select "Git from the command line and also from 3rd-party software".

Verify all three tools

node --version   # must be v18.x or higher
npm --version    # must be 9.x or higher
git --version

All three commands must return version numbers. If any fails, revisit the install step above and open a new terminal after installing.

Allow PowerShell scripts (if blocked) 
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Skip this step if npm --version already runs without error.
Node.js ≥ 18 — node --version returns v18 or higher npm ≥ 9 — npm --version returns 9 or higher Git — git --version returns any version GitHub account — you will authenticate with GitHub OAuth in step 03
02

Install the CLI

Install the current CLI release globally

Install the CLI directly from the GitHub release.

npm install -g https://github.com/NibrasPlatform/nibras-cli/releases/download/v1.0.2/nibras-cli-1.0.2.tgz

Verify: nibras --version should start with v1.0.2.

03

Authenticate with GitHub

Log in with your GitHub account

Run the login command. It will open a browser for GitHub OAuth.

nibras login --api-base-url https://nibras-api.fly.dev

After login, verify with nibras whoami to confirm your session.

04

Set Up a Project

Bootstrap a local project from a project key

Run nibras setup with your project key. This creates the project manifest and starter files.

nibras setup --project your-course/project-key

View the task instructions: nibras task

05

Run Tests Locally

Run the project test suite

Run the configured test command for your project.

nibras test
06

Submit Your Work

Stage, commit, push, and verify

nibras submit runs tests, stages allowed files, creates a commit, pushes to GitHub, and waits for verification.

nibras submit
07

Check Status

Diagnostic commands

Use these commands to check your setup and authentication status at any time.

nibras ping    # check API, auth, GitHub, and project status
nibras whoami  # show signed-in user and linked GitHub account
nibras status  # show recent submission statuses
08

List Courses & Projects

Discover available courses and projects

List your enrolled courses and available projects.

nibras list               # basic listing
nibras list --verbose     # detailed view with milestones
09

Update & Manage CLI

Update, uninstall, or check version

Manage your CLI installation.

nibras update --check                  # check for updates
nibras update --version v1.0.2         # install a specific version
nibras uninstall                        # remove the CLI
10

Troubleshooting

Common issues and fixes

Start with nibras ping — it checks API, auth, GitHub, and project status in one command.

Command not found

Run npm install -g again. If persistent, clear the global link: npm uninstall -g nibras @nibras/cli then reinstall.

EACCES permission denied

Use nvm to manage Node versions. Never use sudo npm install -g.

AUTH_REQUIRED error

Run nibras login --api-base-url again. Your session may have expired.

Repository not found

Ensure your GitHub App is installed. Run nibras ping to check.
